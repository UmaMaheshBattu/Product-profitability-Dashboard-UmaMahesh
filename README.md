# Product-profitability-Dashboard-UmaMahesh
This project presents an interactive dashboard for analyzing product-level profitability using the Nassau Candy Factory dataset. The dashboard helps identify key revenue drivers, profit contribution, and margin performance across products and divisions. It includes visualizations such as bar charts, scatter plots, and treemaps to highlight business insights. The analysis reveals profit concentration among a few products and identifies areas with cost inefficiencies. Users can explore the data using filters like date, division, and product selection. This project demonstrates how data analytics and visualization tools can support better decision-making and improve overall business performance.
## Features
- Product profitability analysis
- Division performance comparison
- Cost vs margin diagnostics
- Profit concentration insights

## Tools Used
- Power BI
- Python (Streamlit)
- DAX

## Dataset
Nassau Candy Factory dataset
